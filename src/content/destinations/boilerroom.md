# Бойлерная — идеальное направление для расстабления!

Тёплая вода, повышенная влажность, жара и живописная плесень на стенах - всё что нужно для идеального отдыха всей семьёй, для пары и для одиночек, ищущих необременяющих знакомств

Разнообразие отелей и кухонь с удобной оплатой - ШансПоинтами по карте и ШансКойном по беспроводной связи. **НЕ НУЖНО БРАТЬ КОШЕЛЁК С СОБОЙ!** Серьёзно, не стоит.

## Особенности направления:

- Термальные и минеральные воды
- Температцрные процедуры
- Уникальная культура
- Экскурсии по канализационным трубам для любителей
